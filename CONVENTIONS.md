# Conventions de code Spira v0.7

## Nommage
- Variables/fonctions : `mots_séparés`
- Constantes : `TOUT_EN_MAJUSCULES`
- Listes : toujours au pluriel

```
projet/
├── principal.sp
├── modules/
├── tests/

```
