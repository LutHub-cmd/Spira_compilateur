# Spira — Guide rapide

## Lancer un programme

```bash
node src/cli.js run principal.sp
node src/cli.js run examples/hello.sp
```

## Lancer les tests

```bash
node src/cli.js run tests/tester_fonctions.sp
```

## Alias recommandé

```bash
alias spira="node /chemin/absolu/vers/spira/src/cli.js"
```

Ensuite : `spira run principal.sp`
